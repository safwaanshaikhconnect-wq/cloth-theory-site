# ClothTheory - Static Landing Page

A minimalist, editorial fashion website for the ClothTheory garment brand.

## Features

- **Responsive Design** - Mobile-first approach with elegant layouts
- **Dark Mode Support** - Theme toggle with localStorage persistence
- **Static Content** - Pure HTML, CSS, and vanilla JavaScript
- **Performance Optimized** - Minimal dependencies for fast load times

## Run Locally

**Prerequisites:** Node.js (v18+)

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

4. Start production server:
   ```bash
   npm start
   ```

The app will be available at `http://localhost:3000`
